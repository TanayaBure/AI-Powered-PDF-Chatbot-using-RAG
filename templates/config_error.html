Configuration Key Required

DocuMind requires cloud AI credentials to run in a serverless environment like Vercel. Please configure the necessary environment variables in your deployment dashboard.

Runtime Error: {{ error_message }}

How to resolve this:

  1. Go to your project settings in the Vercel Dashboard.
  2. Navigate to the Environment Variables tab.
  3. Add the following keys depending on your chosen LLM provider:
    • For HuggingFace (Default): Add HF_TOKEN (Generate a Read token at huggingface.co)
    • For OpenAI: Add LLM_PROVIDER = openai and OPENAI_API_KEY
    • For Groq: Add LLM_PROVIDER = groq and GROQ_API_KEY
  4. Redeploy your project for the keys to take effect.
